## [0.1.0] - 2025-07-10

### Added
- Initial public release of AUDIT.
- Univariate, multivariate, segmentation matrix error, single, pairwise and multi model performance, longitudinal 
and subject analysis modes.
- Support for CSV-based input and automatic parsing of file paths.
- Visualization of segmentations via ITK-SNAP external links.
- Plug-and-play interface using Streamlit.
- Support for `.nii` and `.nii.gz` medical image formats.
- Sample datasets and example configuration files.
